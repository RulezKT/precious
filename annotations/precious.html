This is a convenience script. Use eden(1) for greater convenience.

It takes a string of json, and calls precious.py with it. Only one argument expected. Anything extra is ignored. Run precious json or visit precious-json(7) for format details. The latter (web documentation) is better. Anything less (i.e. nothing) means the convenient defaults will be used.

man = (page) ->
  exec "man #{page}", (err, out) ->
    if err
      console.error "Can't get: `man #{page}`."
      console.error "Go to http://astrolet.github.com/precious/ for help."
      process.exit(1)
    else
      console.log out
